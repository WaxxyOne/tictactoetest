# Tic-Tac-Toe Windows App

A simple desktop Tic-Tac-Toe game built with Python's Tkinter GUI toolkit.

## Requirements

- Python 3.10+
- Tkinter (included with most standard Python installs on Windows)

## Run

```bash
python tic_tac_toe_windows_app.py
```

## Features

- 3x3 clickable board
- Two-player turn tracking (X and O)
- Automatic win and draw detection
- Highlighted winning row/column/diagonal
- "New Game" button to reset the board
